# while y do while

## while

Estructura básica de un while en C:

```c
while (condición)
{
  sentencia;
}
```

Ejemplo práctico:

```c
int i = 0;

while (i < 10)
{
    i++;
    printf("Recorremos este bucle 10 veces. Estamos en la vuelta %d", i);
}
```

## do while

Estructura básica de un do while en C:

```c
do
{
  sentencia;
}
while (condición);
```

Ejemplo práctico:

```c
int i = 0;

do
{
    i++;
    printf("Recorremos este bucle 10 veces. Estamos en la vuelta %d", i);
}
while (i < 10);
```
